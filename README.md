# 🚢 Titanic Survival Dashboard

An interactive Streamlit dashboard built using the Titanic dataset for data analysis and visualization.

---

## 📌 Project Overview

This project focuses on:
- Data Cleaning & Preprocessing
- Exploratory Data Analysis
- Interactive Dashboard Development

Users can upload a cleaned Titanic CSV dataset and explore dynamic visualizations.

---

## 🚀 Features

✅ Upload CSV dataset  
✅ Interactive sidebar filters  
✅ Dataset overview metrics  
✅ Survival analysis charts  
✅ Age distribution visualization  
✅ Correlation heatmap  
✅ Missing value analysis  

---

## 🛠 Technologies Used

- Python
- Pandas
- Streamlit
- Matplotlib
- Seaborn

---

## ▶️ Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Streamlit app:

```bash
streamlit run app.py
```
